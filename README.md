## 🔌 **What's Involved?**

### **📱 Hardware Connection (8 wires)**
```
YELLOW  → Clock Signal
BLUE    → Chip Select
ORANGE  → Data In
RED     → Data Out
BROWN   → Reset
PURPLE  → Data/Command
BLACK   → Ground
PURPLE  → 3.3V Power
```
